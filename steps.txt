1.Install the latest version of Python3

2.Create account in mailtrap.io then you will get credentails of username and password

3.Copy this credenatials and add in env script using the export command:
export MAILTRAP_EMAIL=YOUR MAILTRAP USERNAME
export MAILTRAP_PASSWORD=YOUR MAILTRAP PASSWORD

4.Run the installation script:
./install.sh

5.Activate the virtual environmnet:
source keylogger-env/bin/activate

6.Run the keylogger script:
python keylogger.py
